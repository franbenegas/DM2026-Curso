# Requisitos

Los siguientes requeriminetos son necesarios para poder tomar el curso. 
En caso de tener dudas, por favor contactarme!

:::{warning} Materias obligatorias
Análisis 1, Algebra Lineal.
:::

:::{tip} Materias recomendadas
Alguna materia con modelado basado en ecuaciones diferenciales (Ecuaciones Diferenciales, Física 1). Optimización. Introducción a la estadística y ciencias de datos o Probabilidad. Análisis numérico. Conocimiento básico de programación (Julia, Python o R).
:::
